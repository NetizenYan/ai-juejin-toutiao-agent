# Code of Conduct

## Our Pledge

We want this project to be a respectful learning space for Agent, RAG, OCR, and backend engineering.

Participants are expected to:

- be respectful and constructive
- assume good intent
- explain technical disagreements with evidence
- avoid harassment, personal attacks, and discriminatory language
- avoid posting secrets, private data, or copyrighted full-text datasets

## Unacceptable Behavior

Unacceptable behavior includes:

- harassment or abusive comments
- publishing another person's private information
- posting API keys, passwords, cookies, tokens, or private keys
- encouraging unsafe scraping, unauthorized access, or credential misuse
- using the project to provide financial advice or deterministic stock predictions

## Enforcement

Maintainers may remove comments, close issues, block users, or reject contributions that violate this code of conduct.

## Scope

This applies to repository discussions, issues, pull requests, documentation, and other project spaces.
